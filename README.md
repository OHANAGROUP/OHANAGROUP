## 👤 Sobre Mí

Soy Ingeniero de Software, Arquitecto de Sistemas y CTO enfocado en crear **ecosistemas de software soberanos**. Me especializo en diseñar e implementar agentes autónomos de código abierto que corren localmente, eliminando los costos de inferencia y asegurando la privacidad absoluta de los datos corporativos.

Con más de 6 años de experiencia full-stack, combino el desarrollo de interfaces fluidas de usuario con backends de alta disponibilidad, criptografía de nivel militar y automatización web avanzada capaz de sortear restricciones complejas.

## 💡 Pilares Tecnológicos

*   **🤖 IA Soberana (Sovereign AI):** Orquestación de LLMs multi-agente en hardware local (Ollama, Llama, Qwen). Optimización de latencias en RAM y concurrencia por hardware para despliegues de costo cero.
*   **🔌 Ecosistema MCP:** Creación de servidores y puentes utilizando el **Model Context Protocol (MCP)** para dotar a los modelos de lenguaje de capacidades bidireccionales seguras con bases de datos y sistemas operativos.

## 🚀 Proyectos Destacados (Sovereign Showcase)

### 🔌 [ALPA Hub](https://github.com/ppalominos) — Model Context Protocol (MCP) Bridge
Un puente de código abierto diseñado bajo la especificación **MCP (Model Context Protocol)** que permite a los Modelos de Lenguaje (LLMs) interactuar, auditar e inyectar datos directamente en flujos de trabajo locales mediante protocolos stdio e interfaces SQLite seguras.
*   **Stack:** Node.js, MCP SDK, JSON-Schema, PowerShell, SQLite.
*   **Hito Técnico:** Creación de una pasarela semántica hermética que permite a modelos remotos automatizar contabilidad y reporteo financiero local de forma estructurada y con validaciones estrictas.

### 🤖 **AutomatizAI Hub** — Core AI & DevOps Architect
Motor asíncrono y sistema multi-agente centralizado que coordina tareas operativas en hardware local.
*   **Stack:** FastAPI, Next.js, Supabase, Docker, Ollama (Llama 3.2 & Qwen 2.5).
*   **Hito Técnico:** Desarrollo del worker asíncrono `Obrero Soberano` y mitigación de costos de inferencia en un **99.9%** mediante configuraciones de concurrencia y caché inteligente de hilos en CPU.

### 💰 **Tramita / Dejadwebiar** — Lead Full-Stack & Criptografía
Plataforma SaaS chilena para el procesamiento automatizado de impuestos y trámites a gran escala.
*   **Stack:** Next.js (App Router), Supabase (PostgreSQL), Playwright, MercadoPago API.
*   **Hito Técnico:** Implementación de cifrado AES-256-GCM para llaves gubernamentales aislado por RLS, validación de webhooks con HMAC SHA-256 y un motor de Playwright con caché LRU capaz de evadir bloqueos complejos.

### 🏫 **MVO (Mapa Vivo Operativo)** — Real-Time Digital Twin Architect
Un gemelo digital en tiempo real para la gestión visual e interactiva de infraestructura y telemetría de recintos educacionales.
*   **Stack:** React, Vite, FastAPI, PostgreSQL (Supabase RPC & Triggers), Server-Sent Events (SSE).
*   **Hito Técnico:** Arquitectura de baja latencia usando flujos SSE nativos mapeados sobre deltas de base de datos en tiempo real, soportando múltiples conexiones concurrentes sin pérdida de sincronía.
